#MoonPDF说明文档
-----------------
>一、开发语言  
>本项目使用C++开发语言，使用MFC的OCX开发。  
>二、开发环境  
>项目采用VS2015进行开发  
>三、第三方库  
>第三方库使用开源的mupdf，库版本使用mupdf-1.15.0，库的下载地址为：<https://mupdf.com/downloads/>  
>四、项目编译  
>通过git将项目克隆下来直接用vs进行编译即可。
